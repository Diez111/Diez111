## Sobre mí

Desarrollo software porque me gusta resolver problemas concretos. Trabajo principalmente en **Rust** y **Kotlin**, con algo de TypeScript, Python y Go cuando el proyecto lo pide.

Lo que hago cae, más o menos, en tres áreas:

- **Herramientas de escritorio y mobile** — apps nativas en Rust (GTK) y Android en Kotlin.
- **Automatización e IA** — scripts en Python, pipelines con LLM, scrapers.
- **Sistemas** — scripts para Linux, herramientas de diagnóstico, embebidos.

No me interesa llenar el perfil de buzzwords. Si querés ver qué sé hacer, mirá los repos.

## Repos

Los repos que más me representan:

### [Grafito](https://github.com/Diez111/Grafito)
Rust · wgpu · egui

Aplicación de geometría, álgebra, estadística y cálculo. Es el proyecto al que más tiempo le dediqué: renderer con `wgpu`, UI con `egui`, todo en Rust. Inspirado en GeoGebra pero nativo.

### [career-ops-ar](https://github.com/Diez111/career-ops-ar)
JavaScript · Go

Pipeline de búsqueda laboral con varios agentes LLM. Evalúa ofertas, arma CVs optimizados para ATS, escanea portales. Hecho para el mercado argentino, en español.

### [jds6600-gtk](https://github.com/Diez111/jds6600-gtk)
Rust · GTK4

App de escritorio para controlar un generador de funciones JDS6600 por USB. Detección automática del dispositivo, modo claro/oscuro. Mi primer proyecto serio con GTK4.

### [CanalSPDS](https://github.com/Diez111/CanalSPDS)
Kotlin · Android

Herramienta de diagnóstico para impresoras Samsung y HP que usan el protocolo SPDS. Android nativo, sin frameworks intermedios.

### [Stoic-Launcher](https://github.com/Diez111/Stoic-Launcher)
Kotlin

Launcher para Android, minimalista, escrito en Kotlin. Lo uso a diario.
