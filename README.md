## Sobre mí

Desarrollo software porque me gusta resolver problemas concretos. Trabajo principalmente en **Rust** y **Kotlin**, con TypeScript, Python y Go cuando el proyecto lo pide.

Lo que hago cae, más o menos, en tres áreas:

- **Escritorio y mobile nativo** — apps en Rust (GTK) y Android en Kotlin.
- **Automatización e IA** — scripts en Python, pipelines con LLM, scrapers.
- **Sistemas** — scripts para Linux, herramientas de diagnóstico.

Si querés ver qué sé hacer, mirá los repos.
